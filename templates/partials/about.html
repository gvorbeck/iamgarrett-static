This site has gone through dozens of iterations in the 10+ years that I've owned this domain. At first it was the home of a musical project I worked on, but as priorities shifted, it became my professional homebase. Each time I introduce a new version of this site I hope to leverage new practices and technologies to ensure a modern experience across devices.

Right now, this static single-page site was generated using the Gulp-powered Armadillo generator. Markup was created with the Nunjucks templating language and fed with YAML front matter. Styles were written with the Sass preprocessor. This site's code is housed in this GitHub repository and is built by Travis CI.
